<Tabs>
  <Tab title="API Key">
    Please follow these two simple steps to integrate the Foundation models from OpenAI, Anthropic, and Gemini.
    <Steps>
        <Step title="Get API Key">
            Navigate to `Settings` and click on "Get API Key." This will redirect you to the model provider's webpage where you can generate your API key.
            ![Settings > AI Model Provider > Add API Key](images/app/settings/api_key.png)
        </Step>
        <Step title="Validate & Save">
            The app will automatically validate the API key. Once validated, it will be saved for future use.
            ![Settings > AI Model Provider > Add API Key](images/app/settings/api_key_saved.png)
        </Step>
    </Steps>
  </Tab>
